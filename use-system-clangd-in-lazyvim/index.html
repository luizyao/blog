在 LazyVim 中使用系统 clangd

2024-02-19
#LazyVim
· 阅读量 1000000

Mason LSP 使用系统已安装的 clangd。

新建 ~/.config/nvim/lua/plugins/mason.lua 文件，写入如下内容：

return {
  "neovim/nvim-lspconfig",
  opts = {
    servers = {
      clangd = { mason = false },
    },
  },
}
